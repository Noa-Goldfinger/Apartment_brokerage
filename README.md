# Apartment brokerage 
### Entities:
- An apartment.
- An apartment owner.
- A client.

### An apartment routes mapping
Retrieving the list of apartments.
Retrieving an apartment by ID.
Adding an apartment.
Apartment update.
Status update.
>I chose not to map Route to delete an apartment. Instead I added Route to update status.

### An apartment owner routes mapping
Retrieving the list of owner  apartments.
Retrieving an owner apartment by ID.
Adding an owner  apartment.
Owner apartment update.
Owner apartment deletion.

### A client routes mapping
Retrieving the list of clients.
Retrieving a client by ID.
Adding a client.
Client update.
Client deletion.
